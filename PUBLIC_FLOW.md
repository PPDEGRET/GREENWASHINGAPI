# GreenCheck Public Flow

This document outlines the user flow for both anonymous and authenticated users in the GreenCheck application.

## 1. Anonymous User Flow

The core GreenCheck analysis tool is available to everyone without requiring an account.

### 1.1. Using the Tool

1.  A new user visits the GreenCheck website.
2.  The main analysis tool is immediately available.
3.  The user can upload an ad creative and receive a greenwashing risk analysis.

### 1.2. Usage Limits

-   Anonymous users are subject to the same usage limits as non-premium authenticated users (currently **3 analyses per day**).
-   Usage is tracked based on the user's **IP address**.

### 1.3. Premium Features

-   Premium features are disabled for anonymous users.
-   If an anonymous user attempts to access a premium feature, a modal will appear prompting them to **log in or register** to unlock the feature.

## 2. Authenticated User Flow

By creating an account, users can unlock additional features and a personalized experience.

### 2.1. Registration and Onboarding

1.  A user can choose to register for an account at any time.
2.  After registering and logging in for the first time, the user is required to complete a brief onboarding questionnaire.
3.  This information is used to personalize the analysis results.

### 2.2. Authenticated Experience

-   Once logged in, the user's analysis history is saved to their account.
-   If the user is a **premium member**, all premium features are unlocked, and usage limits are removed.
